# CyberOpsPlanner Dependencies
# Complete list of runtime, development, and system requirements

---

## SYSTEM REQUIREMENTS

**Operating System:**
- Linux (Ubuntu 20.04+, CentOS 8+)
- macOS (10.15+)
- Windows 11 / Windows Server 2019+

**Node.js Runtime:**
- Node.js >= 14.0.0 (tested with 16.x and 18.x)
- npm >= 6.0.0 (comes with Node.js)

**Browser Requirements (Dashboard):**
- Chrome/Chromium 90+
- Firefox 88+
- Safari 14+
- Edge 90+
- Modern browser with ES6+ support

**Disk Space:**
- ~500 MB for Node.js and node_modules
- ~100 MB for project files and operations
- Additional space for operation artifacts (varies)

## INSTALLATION QUICK START

```bash
# 1. Verify Node.js installed
node --version  # Should be >= 14.0.0
npm --version   # Should be >= 6.0.0

# 2. Install project dependencies
cd CyberOpsPlanner
npm install

# 3. Verify installation
npm list  # Shows installed packages

# 4. Start server
npm start  # Or: node server.js

# 5. Verify running
curl http://localhost:3000  # Should return HTML
```

## TROUBLESHOOTING DEPENDENCY ISSUES

**"Module not found" errors:**
```bash
# Reinstall dependencies
rm -rf node_modules package-lock.json
npm install
```

**Port 3000 already in use:**
```bash
# Find and kill process on port 3000
lsof -i :3000
kill -9 [PID]

# Or use different port
PORT=8080 npm start
```

**Node.js version mismatch:**
```bash
# Check current version
node --version

# Install correct version (using nvm if available)
nvm install 16.x
nvm use 16.x
```
